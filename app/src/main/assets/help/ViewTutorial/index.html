その他の情報

新機能

  1. 開発者に寄付する項目を作成しました。
  2. 路線時刻表の列車を列車種別で絞り込む機能を追加しました。詳細はこちら
  3. 自動バックアップ機能：メニューで一番上にある路線はアプリ終了時に自動でバックアップされます。アプリ再起動時にバックアップファイルは自動的に読み込まれます。
  4. ダイヤ、路線をまたいだ列車のコピーが可能になりました。（コピー元とコピー先の路線駅数は同じである必要があります。）
  5. 路線の切り出し・組み入れ機能を搭載しました。駅編集画面から、路線の切り出し・組み入れダイアログを開く事ができます。
